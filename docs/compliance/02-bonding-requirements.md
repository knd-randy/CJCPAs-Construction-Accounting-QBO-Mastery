# Bonding Requirements for Construction Contractors

Construction bonds are critical for contractors pursuing public work and larger private projects. Understanding how bonds work, what sureties evaluate, and how to maximize your bonding capacity can significantly impact your business growth.

## Understanding Construction Bonds

### What is a Surety Bond?

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    SURETY BOND STRUCTURE                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  A surety bond is a THREE-PARTY agreement:                                  │
│                                                                              │
│                         ┌─────────────┐                                     │
│                         │   SURETY    │                                     │
│                         │  (Bonding   │                                     │
│                         │  Company)   │                                     │
│                         └──────┬──────┘                                     │
│                                │                                             │
│              Guarantees        │        Issues bond to                      │
│              performance       │        protect                              │
│                                │                                             │
│                    ┌───────────┴───────────┐                                │
│                    │                       │                                │
│                    ▼                       ▼                                │
│           ┌──────────────┐        ┌──────────────┐                         │
│           │  PRINCIPAL   │        │   OBLIGEE    │                         │
│           │ (Contractor) │◄──────►│   (Owner)    │                         │
│           └──────────────┘        └──────────────┘                         │
│                        Contract for                                         │
│                        construction                                         │
│                                                                              │
│  PRINCIPAL: The contractor who purchases the bond                           │
│  OBLIGEE: The project owner protected by the bond                          │
│  SURETY: The company guaranteeing contractor performance                   │
│                                                                              │
│  KEY DISTINCTION FROM INSURANCE:                                            │
│  Insurance: Pays claims, insured has no obligation to repay                │
│  Surety Bond: If surety pays a claim, contractor must REIMBURSE surety     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Why Bonds Matter

```
BONDS ARE REQUIRED FOR:
─────────────────────────
• Federal projects (Miller Act - over $150,000)
• Most state/local public projects (Little Miller Acts)
• Many private projects over certain thresholds
• Projects with lender requirements

BUSINESS IMPACT:
─────────────────
Without Bonds:               With Bonds:
• Limited to small jobs      • Access larger projects
• Private work only          • Public work available
• Fewer opportunities        • More opportunities
• Lower profit potential     • Higher profit potential
• Less credibility          • Enhanced credibility

EXAMPLE:
Company A: $500K bonding capacity
  → Bids on jobs up to $500K
  → Limited competition pool

Company B: $5M bonding capacity
  → Bids on jobs up to $5M
  → 10x more project opportunities
  → Often better margins on larger work
```

## Types of Construction Bonds

### The "Big Three" Contract Bonds

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    CONTRACT SURETY BONDS                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. BID BOND                                                                 │
│  ─────────────────────────────────────────────────────────────────────────  │
│  Purpose: Guarantees contractor will enter contract if awarded bid          │
│  Amount: Typically 5-10% of bid amount                                      │
│  Duration: Until contract is signed                                         │
│  Cost: Usually free (surety wants the P&P bond)                            │
│                                                                              │
│  If contractor withdraws after winning bid:                                 │
│  → Surety pays owner the difference between winning bid                    │
│    and next lowest bidder (up to bond amount)                              │
│                                                                              │
│  2. PERFORMANCE BOND                                                         │
│  ─────────────────────────────────────────────────────────────────────────  │
│  Purpose: Guarantees contractor will complete the work per contract        │
│  Amount: 100% of contract value                                             │
│  Duration: Until project completion + warranty period                       │
│  Cost: Part of combined P&P premium                                        │
│                                                                              │
│  If contractor defaults:                                                    │
│  → Surety may: Finance contractor to complete                              │
│               Hire new contractor to complete                              │
│               Pay owner to hire completion contractor                      │
│               Pay owner damages up to bond amount                          │
│                                                                              │
│  3. PAYMENT BOND                                                             │
│  ─────────────────────────────────────────────────────────────────────────  │
│  Purpose: Guarantees contractor will pay subs, suppliers, workers          │
│  Amount: 100% of contract value                                             │
│  Duration: Until all claimants are paid                                    │
│  Cost: Part of combined P&P premium                                        │
│                                                                              │
│  If contractor doesn't pay:                                                 │
│  → Subs/suppliers can make claim against the bond                         │
│  → Surety pays claimants                                                   │
│  → Contractor must reimburse surety                                        │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Other Bond Types

```
ADDITIONAL CONSTRUCTION BONDS:

MAINTENANCE/WARRANTY BOND:
─────────────────────────
• Guarantees work for specified period after completion
• Typically 1-2 years
• Usually 10-25% of contract value
• May be separate or included in performance bond

SUBDIVISION/SITE IMPROVEMENT BOND:
─────────────────────────────────
• Guarantees completion of public improvements
• Required by municipalities for subdivisions
• Roads, utilities, sidewalks, etc.
• Released when improvements accepted

LICENSE/PERMIT BOND:
────────────────────
• Required to obtain contractor's license
• Guarantees compliance with regulations
• Protects consumers from contractor fraud
• Amount varies by state/license type

SUPPLY BOND:
────────────
• Guarantees delivery of materials
• Used by material suppliers
• Protects buyer from non-delivery

RETENTION BOND:
───────────────
• Substitutes for cash retention
• Allows contractor to receive full payment
• Surety guarantees completion
• Cost may be less than financing retention
```

### Key Financial Metrics

```
WHAT SURETIES LOOK FOR IN FINANCIALS:

WORKING CAPITAL:
────────────────
Definition: Current Assets - Current Liabilities
Target: 10% of annual revenue (minimum)
Why: Ability to fund job costs before payment received

Working Capital Ratio: Current Assets / Current Liabilities
Target: 1.3 or higher
Why: Buffer for cash flow fluctuations

EQUITY (NET WORTH):
───────────────────
Definition: Total Assets - Total Liabilities
Target: Positive and growing
Why: Ultimate financial backstop

Debt-to-Equity: Total Liabilities / Net Worth
Target: Under 3:1
Why: Indicates leverage/risk level

PROFITABILITY:
──────────────
Gross Margin: (Revenue - COGS) / Revenue
Target: Industry varies, consistency matters
Why: Shows job-level profitability

Net Margin: Net Income / Revenue
Target: 3-8% for healthy contractor
Why: Shows overall profitability

BACKLOG:
────────
Definition: Contracted work not yet completed
Ideal: 6-18 months of revenue
Why: Shows pipeline and sustainability

OVERBILLING/UNDERBILLING:
─────────────────────────
Overbilling: Generally positive (good cash position)
Underbilling: Concerning if excessive (cash drain)
Why: Indicates billing practices and cash management
```

## Calculating Bonding Capacity

### General Guidelines

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    BONDING CAPACITY GUIDELINES                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  RULE OF THUMB (varies by surety and circumstances):                        │
│                                                                              │
│  SINGLE JOB LIMIT:                                                          │
│  Working Capital × 10 = Approximate single job capacity                     │
│                                                                              │
│  Example:                                                                    │
│  Working Capital: $500,000                                                  │
│  Single Job Limit: $500,000 × 10 = $5,000,000                              │
│                                                                              │
│  AGGREGATE LIMIT:                                                            │
│  Single Job Limit × 2-3 = Approximate aggregate capacity                   │
│                                                                              │
│  Example:                                                                    │
│  Single Job: $5,000,000                                                     │
│  Aggregate: $10,000,000 - $15,000,000                                       │
│                                                                              │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  FACTORS THAT INCREASE CAPACITY:                                            │
│  • Strong personal guarantees with liquid assets                           │
│  • Excellent track record                                                   │
│  • CPA-audited financials                                                  │
│  • Strong banking relationship                                              │
│  • Conservative over-billing                                               │
│  • Experienced management team                                              │
│                                                                              │
│  FACTORS THAT DECREASE CAPACITY:                                            │
│  • Under-billing (negative working capital impact)                         │
│  • Losses on financial statements                                          │
│  • Limited experience at proposed project size                             │
│  • Weak personal financial statements                                      │
│  • Past bond claims                                                         │
│  • High debt levels                                                         │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### GAAP Requirements for Bonding

```
PERCENTAGE OF COMPLETION METHOD:

Sureties expect contractors to use percentage-of-completion
(now ASC 606 over-time recognition) for financial statements.

WHY:
• Shows true earned revenue
• Matches costs to revenue
• Reveals job profitability issues
• Standard in the industry

COMPLETED CONTRACT METHOD:
• Generally NOT acceptable for bonding
• Hides profitability issues
• Cash basis even worse
• May be OK for tax, not for bonding

KEY GAAP REQUIREMENTS:
• WIP schedule supporting revenue recognition
• Over/under billing on balance sheet
• Consistent application
• Appropriate cost estimates
• Loss recognition when identified
```

### What NOT to Do

```
BONDING CAPACITY KILLERS:

❌ EXCESSIVE OWNER DISTRIBUTIONS
   Problem: Drains working capital
   Example: $300K profit, $280K distribution = only $20K retained
   Fix: Limit distributions to what business can afford

❌ AGGRESSIVE UNDER-BILLING
   Problem: Reduces working capital, shows cash flow issues
   Example: $500K more earned than billed = $500K financing customers
   Fix: Bill promptly and fully

❌ WEAK PERSONAL FINANCIALS
   Problem: No backup for indemnity
   Example: All net worth in illiquid real estate
   Fix: Maintain liquid assets, improve personal balance sheet

❌ TAKING ON PROJECTS TOO LARGE
   Problem: Execution risk, capacity strain
   Example: $1M contractor bidding $5M job
   Fix: Grow incrementally (50-100% steps)

❌ POOR COMMUNICATION WITH SURETY
   Problem: Surprises destroy trust
   Example: Surety learns of job problems from owner
   Fix: Communicate issues early and honestly

❌ FINANCIAL STATEMENT PROBLEMS
   Problem: Can't rely on numbers
   Example: Cash basis, internal preparation, late
   Fix: CPA-prepared, GAAP, timely
```

### WIP Red Flags for Sureties

```
RED FLAGS SURETIES WATCH FOR:

1. FADE (PROFIT EROSION)
   Original estimate: 20% profit
   Current estimate: 8% profit
   → Why did profit fade? Is it continuing?

2. JOBS OVER 100% COMPLETE
   Costs exceed original estimate
   → Likely a loss situation

3. SIGNIFICANT UNDER-BILLING
   Earned much more than billed
   → Cash flow concern, collection risk

4. STALE RECEIVABLES
   Retention > 1 year old
   → Collection issues?

5. CONCENTRATION
   One job = 50%+ of backlog
   → High risk if that job fails

6. BACKLOG GAPS
   No new work coming in
   → Sustainability concern

7. CONSISTENT LOSSES
   Multiple jobs showing losses
   → Estimating or execution problems
```
