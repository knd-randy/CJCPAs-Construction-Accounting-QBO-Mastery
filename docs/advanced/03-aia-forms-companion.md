# AIA Forms Companion Guide: G702 & G703

## Overview

The American Institute of Architects (AIA) forms are the industry standard for construction billing on commercial projects. Understanding these forms and how they integrate with QuickBooks Online is essential for any contractor working on commercial or institutional projects.

This guide covers:
- Understanding G702 (Application for Payment)
- Understanding G703 (Continuation Sheet/Schedule of Values)
- How to create matching invoices in QBO
- Reconciling AIA billing to QBO
- Common mistakes and how to avoid them

---

### G702 Line-by-Line Explanation

| Line | Description | Source |
|------|-------------|--------|
| **Line 1** | Original contract amount | Per signed contract |
| **Line 2** | Approved change orders (net) | Total approved COs |
| **Line 3** | Current contract sum | Line 1 + Line 2 |
| **Line 4** | Total work completed + stored materials | From G703 Column G total |
| **Line 5** | Retention held | Calculated per contract terms |
| **Line 6** | Total earned less retention | Line 4 - Line 5 |
| **Line 7** | Previous payments certified | Prior G702 Line 6 |
| **Line 8** | Current payment due | Line 6 - Line 7 |
| **Line 9** | Balance to finish | Line 3 - Line 6 |

### G703 Column Definitions

| Column | Name | Description |
|--------|------|-------------|
| **A** | Scheduled Value | Contract value for this line item |
| **B** | Work Completed (Previous) | Total billed in prior pay apps (Columns D+E) |
| **C** | Work Completed (This Period) | New work completed this billing period |
| **D** | Materials Stored | Materials on-site but not installed |
| **E** | Total Completed & Stored | Sum of B + C + D |
| **F** | Percentage Complete | Column E ÷ Column A |
| **G** | Balance to Finish | Column A - Column E |
| **H** | Retainage | Retention held (if shown separately) |

## Creating the Schedule of Values (SOV)

### What Is an SOV?

The Schedule of Values is the agreed-upon breakdown of the contract into line items for billing. It's created at project start and approved by the architect/owner before first billing.

### SOV Best Practices

**DO:**
- Break down into meaningful work packages
- Front-load general conditions (they're incurred early)
- Include line items that allow progress measurement
- Match subcontractor values where possible
- Include change order lines as they're approved

**DON'T:**
- Over-inflate early items (architects will catch it)
- Create too few lines (hard to show progress)
- Create too many lines (administrative burden)
- Include lump-sum items you can't measure

```

  02 01 - Demolition                           $30,000           1.2%

  02 04 - Paving & Sidewalks                   $40,000           1.6%
```

## Creating QBO Invoices to Match AIA Forms

### Method 1: Simple Approach (Recommended)

Create a QBO invoice that summarizes the G702/G703:

**Invoice Line Items:**
```
1. Progress Billing - Application #5
   (Per attached AIA G702/G703)                    $350,000.00

2. Less: Retainage (10%)                          ($35,000.00)

   TOTAL DUE THIS APPLICATION                     $315,000.00
```

**Pros:** Simple, clean, matches check amount
**Cons:** Less detail in QBO

### Method 2: Detailed Approach

Create line items matching your SOV:

**Invoice Line Items:**
```
01 - General Conditions (This Period)              $15,000.00
04 - Masonry (This Period)                         $25,000.00
05 - Structural Steel (This Period)                $20,000.00
06 - Carpentry (This Period)                       $30,000.00
07 - Roofing (This Period)                         $50,000.00
08 - Doors/Windows/Glass (This Period)             $60,000.00
09 - Drywall (This Period)                         $40,000.00
10 - Flooring (This Period)                        $25,000.00
12 - Mechanical (This Period)                      $45,000.00
13 - Plumbing (This Period)                        $35,000.00
14 - Electrical (This Period)                      $55,000.00
15 - Fire Protection (This Period)                 $30,000.00
CO1 - Added Electrical (This Period)               $10,000.00
CO2 - HVAC Upgrade (This Period)                   $10,000.00
─────────────────────────────────────────────────────────────
Gross Billing This Period                         $450,000.00
Less: Retainage (10%)                            ($45,000.00)
─────────────────────────────────────────────────────────────
TOTAL DUE THIS APPLICATION                        $405,000.00
```

**Pros:** Detail in QBO, revenue by category
**Cons:** More data entry, must match exactly

### Method 3: Products/Services Approach

Create Products/Services items for each SOV line:

**Setup:**
1. Create Service items: "SOV-01 General Conditions", "SOV-02 Sitework", etc.
2. Each maps to appropriate revenue account
3. Invoice using these items each period

### QBO Setup for AIA Billing

**Accounts Needed:**
```
BALANCE SHEET
12100 Accounts Receivable - Progress Billings
12200 Accounts Receivable - Retention
       (or use sub-customer for retention tracking)

INCOME
40100 Contract Revenue - Progress Billing
40200 Contract Revenue - Change Orders
```

**Recording the Invoice:**
```
Invoice #: APP-005
Customer: Office Building Project
Date: November 30, 2024

Line 1: Progress Billing App #5         $350,000    (40100)
Line 2: Retention Held (10%)           ($35,000)    (40100 or contra)

Total: $315,000
```

### Common Reconciliation Issues

| Issue | Cause | Solution |
|-------|-------|----------|
| AR doesn't match | Invoices not matching G702 | Review invoice amounts |
| Retention off | Not tracking retention properly | Set up retention tracking |
| Revenue mismatch | Timing differences | Ensure period-end cutoff |
| CO not included | Change order not invoiced | Add CO to invoice |

## Stored Materials

### What Are Stored Materials?

Materials delivered to jobsite (or off-site storage) but not yet installed.

**Example:**
- Windows delivered, stored in warehouse
- Can bill for windows (Column D on G703)
- Release to Column E when installed

### Stored Materials Requirements

**To Bill for Stored Materials:**
1. Materials are delivered and inspected
2. Invoiced to you by supplier
3. Properly stored and protected
4. Not at risk of theft or damage
5. Title transfers to owner upon billing
6. Insurance covers stored materials

### QBO Treatment for Stored Materials

**Option 1: Bill Directly**
```
Invoice:
  Stored Materials - Windows         $50,000
  Less: Retention (10%)             ($5,000)
  Net Due                           $45,000
```

**Option 2: Track as Inventory First**
```
When received:
  Debit: Stored Materials Inventory  $50,000
  Credit: Accounts Payable                    $50,000

When billed to owner:
  Debit: Accounts Receivable         $50,000
  Credit: Stored Materials Revenue            $50,000
  (Transfer from inventory to COGS when installed)
```

## AIA Billing Timeline

### Typical Monthly Billing Cycle

```
MONTHLY AIA BILLING CYCLE

Day 1-25: Work Period
├── Work is performed
├── Track costs daily
└── Prepare for billing

Day 25-27: Billing Preparation
├── Field provides % complete by line item
├── Office compiles G703
├── Calculate retainage
└── Prepare G702 summary

Day 27-28: Internal Review
├── PM reviews for accuracy
├── Verify against subcontractor billings
└── Check for change orders to include

Day 28-30: Submission
├── Submit to Architect
├── Architect reviews (may take 7-14 days)
└── Architect certifies and forwards to Owner

Day 30-45 (or longer): Payment
├── Owner processes payment
├── Typically 30 days from certification
└── Watch for payment terms in contract
```

### Contract Payment Terms

| Payment Terms | Meaning |
|---------------|---------|
| **Net 30 from Application** | 30 days from when you submit |
| **Net 30 from Certification** | 30 days from architect approval |
| **Net 45** | 45 days from relevant date |
| **Pay When Paid** | After owner pays GC (controversial) |

## Common AIA Billing Mistakes

### Mistake 1: Front-Loading the SOV

**Problem:** Inflating early items to bill more upfront
**Risk:** Architect rejects SOV; credibility damage
**Solution:** Use realistic values; earn trust

### Mistake 2: Overbilling Work Not Complete

**Problem:** Billing for 80% when only 60% done
**Risk:** Certification rejection; contract termination
**Solution:** Be accurate; round down if uncertain

### Mistake 3: Missing Change Orders

**Problem:** Work done under CO but not billed
**Risk:** Never get paid for the work
**Solution:** Include approved COs in each application

### Mistake 4: Retainage Calculation Errors

**Problem:** Calculating retention incorrectly
**Risk:** Short payment; reconciliation headaches
**Solution:** Verify retention per contract; check math

### Mistake 5: Not Reconciling to QBO

**Problem:** AIA and QBO show different amounts
**Risk:** Cash flow confusion; audit issues
**Solution:** Monthly reconciliation process

## QBO Reports for AIA Billing

### Useful Reports

1. **Open Invoices by Customer**
   - Shows outstanding AR by project
   - Include retention tracking

2. **Customer Balance Detail**
   - Full billing history
   - Application-by-application

3. **Sales by Customer Summary**
   - Total billed per project
   - Compare to contract value

4. **A/R Aging**
   - How long has payment been outstanding?
   - Flag slow payers

### Custom Report: Contract Status

Create a custom report showing:
- Contract value
- Total billed
- Total collected
- Retention held
- Balance to bill

## Key Takeaways

1. **G702 summarizes** - One page overview for payment certification
2. **G703 details** - Line-by-line schedule of values
3. **SOV is critical** - Get it right at the start
4. **QBO must match** - Reconcile monthly
5. **Track retention separately** - It's a significant asset
6. **Include all COs** - Don't leave money on the table
7. **Bill accurately** - Overbilling destroys trust
8. **Follow the timeline** - Late billing = late payment
