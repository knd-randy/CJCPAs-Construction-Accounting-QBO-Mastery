## Quick Rate Calculator

### Hourly Equipment Rate Formula

```
QUICK RATE FORMULA

Internal Hourly Rate =
    (Annual Ownership Costs ÷ Annual Hours)
  + Hourly Operating Costs
  + Markup

WHERE:

Annual Ownership Costs =
  + Depreciation
  + Interest/Financing
  + Insurance
  + Property Tax
  + Storage

Hourly Operating Costs =
  + Fuel
  + Maintenance
  + Repairs
  + Consumables

Markup = 15-25% for overhead and profit
```

## Own vs. Rent Analysis

### Decision Framework

```
OWN VS. RENT ANALYSIS WORKSHEET

Equipment Type: _________________________________
Current Rental Rate: $_________/day  $_________/week  $_________/month

STEP 1: Estimate Annual Usage
────────────────────────────────────────────────────────────────────────────────
  Days needed this year:              _____ days
  Days needed next year (est):        _____ days
  Days needed Year 3 (est):           _____ days
  Average Annual Days:                _____ days

STEP 2: Calculate Rental Cost
────────────────────────────────────────────────────────────────────────────────
  Annual Rental Cost: _____ days × $_____/day = $_____________

STEP 3: Calculate Ownership Cost
────────────────────────────────────────────────────────────────────────────────
  Purchase Price:                               $_____________
  Financing Cost (5 years):                     $_____________
  Insurance (annual):                           $_____________
  Maintenance (annual):                         $_____________
  Storage (annual):                             $_____________
  ─────────────────────────────────────────────────────────────
  5-Year Total Ownership Cost:                  $_____________
  Less: Estimated Resale Value:                ($_____________)
  ─────────────────────────────────────────────────────────────
  Net 5-Year Cost:                              $_____________
  Annual Equivalent:                            $_____________

STEP 4: Compare
────────────────────────────────────────────────────────────────────────────────
  Annual Rent Cost:                             $_____________
  Annual Own Cost:                              $_____________

  DIFFERENCE:                                   $_____________

  □ OWN - If ownership cost is lower
  □ RENT - If rental cost is lower

STEP 5: Consider Intangibles
────────────────────────────────────────────────────────────────────────────────
  □ Availability when needed (own advantage)
  □ Maintenance hassle (rent advantage)
  □ Cash flow impact (rent advantage)
  □ Tax benefits of ownership
  □ Storage space requirements
  □ Utilization risk if work decreases

RECOMMENDATION: □ OWN    □ RENT    □ RENT UNTIL USAGE INCREASES
```

### Break-Even Analysis

```
BREAK-EVEN UTILIZATION

Equipment: _________________________________

Break-Even Days = Net Annual Ownership Cost ÷ Daily Rental Rate

Example:
  Net Annual Ownership Cost:    $15,000
  Daily Rental Rate:            $450
  Break-Even Days:              33 days/year

If you use the equipment MORE than 33 days/year → OWN
If you use the equipment LESS than 33 days/year → RENT
```

### Recording Equipment Charges to Jobs

**Option 1: Products/Services**

Create a Product/Service for each piece of equipment:
- Name: "E-001 Backhoe Charge"
- Type: Service
- Rate: $56.00/hour
- Income Account: 42000 Equipment Income
- Expense Account: 54000 Equipment Costs

**Option 2: Journal Entry**

```
Debit:  WIP - Equipment (or Job Cost - Equipment)    $XXX
Credit: Equipment Revenue (internal)                  $XXX
```

### Tracking Equipment by Project

1. Create each major equipment item as a Product/Service
2. When charging to project, create invoice or time entry
3. Assign to Customer/Project
4. Run "Profit & Loss by Customer" to see equipment charges per job

## Formulas for Excel/Google Sheets

### Hourly Depreciation
```
= (Purchase_Price - Salvage_Value) / Useful_Life_Hours
```

### Annual Ownership Cost
```
= Depreciation + Interest + Insurance + Tax + Storage
```

### Internal Hourly Rate
```
= (Annual_Ownership / Annual_Hours + Hourly_Operating) * (1 + Markup%)
```

### Break-Even Days
```
= Annual_Ownership_Cost / Daily_Rental_Rate
```

### Utilization Percentage
```
= Hours_Used / Available_Hours * 100
```

## Best Practices

1. **Review rates annually** - Costs change, update rates accordingly
2. **Track actual costs** - Compare estimates to reality
3. **Monitor utilization** - Low utilization = consider renting instead
4. **Charge all jobs** - Every project should bear equipment cost
5. **Compare to market** - Stay competitive with rental rates
6. **Include all costs** - Don't forget insurance, storage, financing
7. **Document decisions** - Keep records of own vs. rent analysis
