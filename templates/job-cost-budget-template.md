Use this template to create a detailed cost budget for each project. This budget becomes the baseline for tracking actual costs and calculating WIP percentage complete.

## QBO Integration

### Mapping Budget to QBO Accounts

| Budget Code | QBO Account # | QBO Account Name |
|-------------|---------------|------------------|
| L-01 to L-07 | 51000 | Direct Labor |
| M-01 to M-15 | 52000 | Materials |
| S-01 to S-15 | 53000 | Subcontractors |
| E-01 to E-07 | 54000 | Equipment |
| O-01 to O-12 | 55000 | Other Direct Costs |

### Setting Up in QBO

1. Create project/customer in QBO
2. Use Products/Services matching budget codes
3. Assign all costs to project
4. Run P&L by Customer for actuals
5. Compare to this budget monthly
